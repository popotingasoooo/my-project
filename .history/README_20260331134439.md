# TaskFlow

A Laravel 12 task-management web app with a Kanban-style ToDo board, user and role administration, and activity logging.

---

## ✨ Features

### Task management
- Kanban board with **To Do**, **In Progress**, and **Done** columns
- Create, edit, and delete tasks
- Set **priority**, **due date**, and **assignee**
- Task comments for collaboration
- Staff users can view their assigned tasks and update progress

### Admin tools
- User CRUD with **search**, **role filtering**, and **soft delete / restore**
- Role and permission management with **Spatie Laravel Permission**
- Activity log page with readable change history
- Dashboard with quick stats and recent activity

### Authentication
- Laravel Breeze login flow
- Profile management for authenticated users
- Role-based access control for app sections and actions

## 🚀 Setup

1. **Install PHP dependencies**
   ```bash
   composer install
   ```

2. **Create the environment file**
   ```bash
   cp .env.example .env
   ```

3. **Generate the app key**
   ```bash
   php artisan key:generate
   ```

4. **Configure the database** in `.env`

5. **Run migrations and seeders**
   ```bash
   php artisan migrate --seed
   ```

6. **Install frontend dependencies**
   ```bash
   npm install
   ```

7. **Start the app**

   Option A:
   ```bash
   composer run dev
   ```

   Option B:
   ```bash
   php artisan serve
   npm run dev
   ```

Then visit: `http://localhost:8000`

## 🔐 Permissions used in the app

The project includes these main permissions:

- `manage-users`
- `view-users`
- `manage-roles`
- `view-roles`
- `view-logs`
- `manage-tasks`
- `view-tasks`

Admins are seeded with full access. Staff users are limited to the task workflow and basic viewing access.

## 🧭 App sections

### Dashboard
Shows:
- total users
- total roles
- total activity logs
- task counts and recent activity

### Tasks
- View tasks on the Kanban board
- Create and assign new tasks
- Update status and manage due dates
- Add comments on task detail pages

### Users
- Manage user accounts
- Search and filter by name, email, or role
- Restore soft-deleted users

### Roles
- Create roles
- Assign permissions
- Assign roles to users

### Activity logs
- Review who changed what and when
- View changed attributes such as due dates and assignees

## 🐛 Troubleshooting

### Permissions or roles not updating
```bash
php artisan cache:clear
php artisan config:clear
php artisan db:seed
```

### Frontend styles not showing
```bash
npm install
npm run dev
```

### Production assets need rebuilding
```bash
npm run build
```
